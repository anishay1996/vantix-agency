X-Agency Creative Studio Template

X-Agency is a highly professional, modern, and clean HTML5 template designed for creative agencies, studios, and digital service providers. It features 3 unique home page versions, a portfolio grid, and a functional contact form.

Key Features

  • 3 Unique Home Page Variants (Modern, Minimalist, Cyber)
  • Full Multi-Page Setup (Services, Work, Contact)
  • Fully Responsive Design (Bootstrap 5)
  • Local Asset Hosting (Bootstrap, jQuery, Images)
  • Premium Glassmorphism UI Components
  • Advanced CSS3 Animations & Scroll Reveal
  • Clean and Well-Documented Code

Customization

Changing Colors

You can easily change the primary colors of the template by editing the CSS variables in assets/css/style.css:

:root {
    --primary: #007bff;
    --secondary: #6610f2;
    --gradient: linear-gradient(135deg, #007bff 0%, #6610f2 100%);
    --bg-dark: #0a0a0a;
}

Updating Images

All project images are stored in assets/img/. To replace them, simply add your images to this folder and update the src attribute in the HTML files.
